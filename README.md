# Programa de Suma en Python

Este es un programa sencillo en Python que permite sumar dos números ingresados por el usuario.

## 📌 Funcionalidad

El programa:

1. Solicita dos números al usuario.
2. Realiza la suma de ambos números.
3. Muestra el resultado en pantalla.

## ▶️ Código

```python
# Programa sencillo para sumar dos números

# Pedir los números al usuario
num1 = float(input("Ingresa el primer número: "))
num2 = float(input("Ingresa el segundo número: "))

# Realizar la suma
suma = num1 + num2

# Mostrar el resultado
print("La suma es:", suma)
